1. Estructura de una Etiqueta HTML

Componentes de una Etiqueta

<etiqueta atributo="valor">Contenido</etiqueta>

Partes de una Etiqueta

  • <etiqueta> - Etiqueta de apertura
  • atributo="valor" - Atributos (opcionales)
  • Contenido - Texto o elementos hijos
  • </etiqueta> - Etiqueta de cierre

Ejemplos Prácticos

Este es un párrafo con texto.

Enlace con atributos

Tipos de Etiquetas

  • Etiquetas de bloque: Ocupan todo el ancho disponible (p, div, h1-h6)
  • Etiquetas en línea: Solo ocupan su contenido (span, a, img)
  • Etiquetas void: No tienen contenido ni cierre (img, br, hr)

6. Imágenes y Rutas

Imagen desde URL Externa

Imagen con Atributos

Tipos de Rutas

Ruta Relativa (misma carpeta)

Si la imagen está en la misma carpeta: src="imagen.jpg"

Ruta Relativa (carpeta padre)

Para ir a la carpeta padre: src="../imagen.jpg"

Ruta Relativa (subcarpeta)

Para ir a una subcarpeta: src="imagenes/imagen.jpg"

Ruta Absoluta

Desde la raíz del sitio: src="/imagenes/imagen.jpg"

URL Completa

Desde internet: src="https://ejemplo.com/imagen.jpg"

Imagen con Figura y Leyenda

Imagen Responsive

Atributo alt (Importante para Accesibilidad)

El atributo alt es obligatorio y describe la imagen para:

  • Lectores de pantalla
  • Cuando la imagen no carga
  • SEO (optimización para motores de búsqueda)
